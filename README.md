# Health Prediction System

## Project Overview

The Health Prediction System is a web-based application developed using Flask, SQLite, HTML, and CSS. It allows users to manage patient records and provides basic health-risk predictions based on patient health parameters.

## Features

* Add new patient records
* View patient records
* Edit existing patient information
* Delete patient records
* Health risk prediction based on glucose and cholesterol values
* SQLite database integration

## Technology Stack

* Python
* Flask
* SQLite
* HTML
* CSS

## CRUD Operations

* Create: Add patient records
* Read: View all patient records
* Update: Modify patient information
* Delete: Remove patient records

## Project Structure

* app.py – Flask application
* create_db.py – Database creation script
* templates/ – HTML templates
* static/ – CSS files
* database/ – SQLite database

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open:
   http://localhost:5000
